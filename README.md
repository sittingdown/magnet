## Instructions
**Step 1**:
  - Open magnetfishing.py
  - Tab into FiveM
**Step 2**:
  When in the fishing spot
  - Press E to start the minigame
  - Press F4 to select a waiting point for mouse
  - Click the magnet at the end of the rope
**Step 3**:
  With the minigame open
  - Press F5 and drag from one corner of the water to the other
**Step 4**:
  If you are using ReShade or anything to change pixel colors continue with this step
  - Press F6 and click the Icon you need to press to finish the minigame
**Step 5**:
  Without the minigame on screen
  - Press F3 to start/stop the script

## Info
Best values for Cast Delay is 1.75, Click Wait is 2.80.
Pressing the Pin icon will toggle "Stay on top".
When tabbing out of the game it will Pause the script.
