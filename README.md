## Setup

**Step 1**
- Open `magnetfishing.py`
- Tab into FiveM

**Step 2** - When in the fishing spot
- Press E to start the minigame
- Press F4 to select a waiting point for the mouse
- Click the magnet at the end of the rope

**Step 3** - With the minigame open
- Press F5 and drag from one corner of the water to the other

**Step 4** - If you are using ReShade or anything that changes pixel colors
- Press F6 and click the icon you need to press to finish the minigame

**Step 5** - Without the minigame on screen
- Press F3 to start / stop the script

---

## Hotkeys

| Key | Action |
|-----|--------|
| F3 | Toggle on / off |
| F4 | Set mouse wait spot |
| F5 | Set scan region |
| F6 | Pick target color |
| F7 | Quit |

---

## Info

- Best values: Cast Delay **1.75s**, Click Wait **2.80s**
- Pressing the pin icon will toggle **Stay on top**
- Tabbing out of the game will **pause** the script
- All settings save automatically between sessions
- If a new version is available it will show next to the version number — click it to go to the release
